# Frontendのプロジェクトガイドライン

## プロジェクト概要
WebエンジニアのCV（履歴書＋職務経歴書）の管理やシステム全体に関わる設定等を行うためのユーザーインターフェースを提供する

## 技術スタック
- フレームワーク
  - React 18+ with TypeScript
- UIライブラリ
  - shadcn/ui
- 状態管理
  - Jotai
- ルーティング
  - TanStack Router
- HTTPクライアント
  - ky (github.com/sindresorhus/ky)
- OpenAPIコード生成
  - OpenAPI Generator (openapitools/openapi-generator-cli)
